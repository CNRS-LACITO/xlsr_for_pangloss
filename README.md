This directory code to fine-tune a XLSR model for low-ressource languages from the [Pangloss Collection](https://pangloss.cnrs.fr/).

It is based on the code developed by [Cécile Macaire](https://github.com/macairececile) during her internship at LACITO and has been used for our JEP'22 and ComputEL'5 papers.
